### 参数


```
工具库依赖jquery

width  Number  外层盒子的大小
height Number  外层盒子的高度
type   'fade' (淡入淡出)  'animation' (从左到右轮播)  String
isAuto Boolean 是否自动轮播
arrowStatus 'hover' 'alwayls' 'hide'  箭头的状态
isShowCircle Boolean 是否显示小圆点
content dom对象  需要轮播的内容
times Number 自动轮播时间
circlePosition 'left' 'right' 'center'  String 小圆点的位置
```
